Course Objectives

  • Explain the objectives of information security (Confidentiality, Integrity, Availability - CIA Triad).
  • Understand basic categories of threats and attacks (passive, active).
  • Understand various cryptographic algorithms (Symmetric: DES, AES; Asymmetric: RSA, Diffie-Hellman; Hashing: SHA).
  • Describe public-key cryptosystems and their fundamental ideas.
  • Describe secure software design principles and programming techniques (SSDLC).
  • Discuss Web security concepts and common threats (SQLi, XSS).
  • Understand the role and types of Firewalls and WAFs.
  • Understand Intrusions and intrusion detection concepts (Covered implicitly, expanded in potential future units).
  • (Future Goal) Design Intrusion Detection System IDS (python).

Assignments (Examples from Slides)

  • **Assignment 1 (Unit 2):** Write Python code to decrypt messages using the Caesar cipher.
  • **Assignment 2 (Unit 2):** Encrypt various plaintexts using Caesar, Monoalphabetic, Polyalphabetic, Playfair, Hill, and XOR ciphers; Find SDES keys.
  • **Assignment 3 (Unit 3):** Perform RSA encryption/decryption and Diffie-Hellman key exchange calculations.
  • **Assignment 4 (Unit 4/5):** Explain hash functions & integrity; How digital signatures work; SHA-1 insecurity; Principles of secure software design; Best practices for secure development.
